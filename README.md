## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher




### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/machapraveen/tkinter-dashboards.git
   cd tkinter-dashboards
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**
   ```bash
   python main.py
   ```
